# IMDB Sentiment Analysis

## Project Overview

This project focuses on sentiment classification of IMDB movie reviews using Natural Language Processing (NLP) techniques.

The objective is to classify movie reviews as positive or negative by transforming text data into numerical features using TF-IDF vectorisation and applying Logistic Regression for supervised machine learning.

## Dataset

- IMDB Large Movie Review Dataset
- 50,000 labelled movie reviews
- Balanced positive and negative sentiment classes

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- TF-IDF Vectorisation
- Logistic Regression
- Matplotlib

## Key Tasks

- Text preprocessing
- Feature extraction using TF-IDF
- Model training and evaluation
- Confusion matrix analysis
- Classification performance assessment

## Results

The Logistic Regression model achieved strong performance in classifying movie review sentiment and demonstrated the effectiveness of TF-IDF features for text classification tasks.
